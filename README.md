# Template-Backend
🐱‍👤 Template de backend node com configurações definidas para o Visual Code, ESlint, Prettier e algumas libs padrões.

#### Conteúdo

- Express;
- Nodemon;
- Sucrase;
- Sequelize;
- Configurações ESLint e Prettier
- Estrutura de diretório básico
